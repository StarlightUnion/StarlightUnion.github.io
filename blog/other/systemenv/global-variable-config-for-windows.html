    # Windows系统中配置全局变量

    # 安装失败

    今天在Windows中尝试安装yarn失败了。。

    失败过程如下：↓

    • 1.从npm安装yarn，输入命令npm instal -g yarn。

    • 2.接着输入yarn -version查看版本信息，结果提示不能用。。

    首先检查安装过程，用的是管理员权限，不存在权限不足的问题；接着检查安装路径，路径下已经有yarn的文件存在。所以问题应该出在全局变量上。

    和CentOS配置Node.js和Nginx环境一样，想要在全局中使用npm、node、nginx命令需要建立软链接到全局路径下。

    # 配置环境变量

    展开查看进入窗口过程

    快捷进入环境变量窗口编辑两种方式：

    • 1.桌面此电脑右键选择属性 -> 左侧选择高级系统设置 -> 选择第三tab高级 -> 环境变量。

    • 2.快捷键Win + R弹窗输入sysdm.cpl-> 选择第三tab高级 -> 环境变量。

    在环境变量窗口中选择系统变量栏，再选择Path，双击打开编辑窗口，添加全局环境中yarn的地址，地址在yarn安装完成后会显示在命令行中。。
