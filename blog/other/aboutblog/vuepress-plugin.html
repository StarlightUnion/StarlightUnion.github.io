# 3.google-analytics

一个网站分析插件，可以在谷歌分析的后台界面查看博客网站的流量、访问来源、在线人数等信息。

该插件需要使用GAID（即 Google Analytics ID），所以需要谷歌账号并且具备一定的魔法知识。

首先全局安装，官网 (opens new window)推荐使用yarn安装，否则会出问题。

npm install -g yarn # 安装yarn包管理器
yarn --version # 查看yarn版本

yarn add -D @vuepress/plugin-google-analytics
# OR npm install -D @vuepress/plugin-google-analytics

module.exports = {
  ...
  plugins: [
    ['@vuepress/google-analytics', {
        'ga': '' // UA-00000000-0 填上GAID
    }]
  ]
  ...
}

# 5.last-updated

从GitHub获取最后提交时间戳(timestamp)作为最后更新的时间基准。

该插件需要安装moment👇

npm install moment --save   # npm
yarn add moment             # Yarn
1
2

配置

module.exports = {
  ...
  plugins: [
    ...
    ['@vuepress/last-updated', {
        transformer: (timestamp, lang) => {
          // 不要忘了安装 moment
          const moment = require('moment')
          moment.locale(lang)
          return moment(timestamp).fromNow()
        }
      }
    ]
    ...
  ]
}
