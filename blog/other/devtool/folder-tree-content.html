# 自动生成文件目录树总结

在看网络上的不管是文档也好，还是各位大佬的博客也好，会发现里面经常会出现文件目录结构的树状示意图。比如下面这个vuepress的目录结构示意 (opens new window)，这个肯定装了其他插件。那么怎么自动生成一个朴实无华，并且极致简洁😂的这种示意结构？

# 一、windows

windows不需要安装说什么插件，可以直接通过命令行生成，cmd和powershell都可以使用。

tree # 在命令行中生成目录树，只有当前目录中的文件，不包括子文件
tree /F # 生成该目录下所有文件和子文件的目录树

tree > tree.txt # 将生成的目录树保存到tree.txt 下同
tree /F > tree.txt

# 二、macOS

需要安装一个插件，在终端中输入brew install tree，此操作需要安装HomeBrew，可参考此文。

接下来就可以使用命令了，命令和上面的差不多。

tree # 在终端中显示目录树
tree -d # 显示该目录下所有子文件夹的目录树
tree > tree.txt # 将生成的目录树保存到tree.txt
